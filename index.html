WHINNY

View Demo Video

A cross platform app for all things horses

Features:
  • User registration
    • Allows users to sign up for a Whinny Account. This creates a user record in the database and sends them a confirmation text.
  • 1 on 1 chat
  • Push Notifications with Ionic Push
  • Emojis - Using contenteditable div and html parsing
  • Group Chat - Allows users to communicate in large groups
  • Web Client allows an authorized user to log in and
  • Server that handles messaging, photo messaging, group management routes etc, made with Express.js, Node.js
  • Photo uploading and hosting with Amazon S3
  • Works on iOS and Android
Made with: Ionic Angular 1.5 Knex Express Node Postgres Twilio Ionic Push Gulp

BARN ALERT

Barn alert is an app that allows people to receive text notifications about equine contagious disease outbreaks in their state. Currently the EDCC sends email alerts to their email list, but users receive emails for all of North America. The Barn Alert server receives an EDCC email and parses the location. Then, we send text alerts to any user who is subscribed to that state's alerts.

Features:

  • Contagious disease text alerts by state
  • Twitter updates from the EDCC from the twitter API
  • Fully mobile responisve
Made with: React Node Express Postgres Twitter API Twilio React-Bootstrap

This project is online at: http://www.barnalert.com/
